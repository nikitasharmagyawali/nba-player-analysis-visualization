# NBA Player Analysis & Visualization

## Overview
Editorial-style and network-based analysis of NBA player performance using public per-36-minute statistics.

## Visualizations
### Scoring Style Distribution (NYT-Inspired)
Scatterplot comparing two-point vs three-point shot share to highlight differences in player scoring styles.

### Player Teammate Network
Network visualization modeling teammate relationships across seasons, revealing tightly connected player groups and key connectors.

## Data
Public NBA dataset (per-36-minute stats) from Kaggle.

## Tools
Python (Pandas, NetworkX), Tableau
